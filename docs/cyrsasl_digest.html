    is_digesturi_valid/2
    Check if the digest-uri is valid. RFC-2831 allows to provide the IP address in Host, however ejabberd doesn't allow that. If the service (for example jabber.example.org) is provided by several hosts (being one of them server3.example.org), then digest-uri can be like xmpp/server3.example.org/jabber.example.org In that case, ejabberd only checks the service name, not the host.

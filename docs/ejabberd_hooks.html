    run/3
    run_fold/3
    Run the calls of this hook in order. The arguments passed to the function are: [Val | Args]. The result of a call is used as Val for the next call. If a call returns 'stop', no more calls are performed and 'stopped' is returned. If a call returns {stop, NewVal}, no more calls are performed and NewVal is returned.

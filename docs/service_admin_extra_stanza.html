    send_packet_all_resources/3
    Send a packet to a Jabber account. If a resource was specified in the JID, the packet is sent only to that specific resource. If no resource was specified in the JID, and the user is remote or local but offline, the packet is sent to the bare JID. If the user is local and is online in several resources, the packet is sent to all its resources.

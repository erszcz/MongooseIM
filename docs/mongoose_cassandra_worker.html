    write/5
    Execute batch write query to Cassandra (insert, update or delete). Note that Cassandra doesn't like big batches, therefore this function will try to split given rows into batches of 20 rows and will fall back to smaller batches if Cassandra rejects the query due to its size being to big.

    Same as @see write/5 but asynchronous (without response).
